(On OSX, control key (Ctrl) is the Command key (⌘))

Files

  • Ctrl+o : Open file(s)
  • Ctrl+w : Close the selected file(s)
  • Ctrl+r : Reload the selected file(s)
  • Ctrl+d : Duplicate the selected file(s)
  • Ctrl+s : Save the selected file(s)
  • SDIF files can be followed by a selector:
    <filename>[::[#stream][:frame][/matrix][.column]

Views: zoom and scrolling

  • Mouse wheel : Zoom in and out.
  • Shift + Hold right-click + Mouse move: Zoom on x,y-axis according to the mouse move, while keeping the coordinate pointed fixed.
  • Shift + Wheel : Scroll the view in the waveform and spectrogram views.
  • 's' key : Zoom on the current selection.

Selection

  • Hold left-click + Mouse move: Create a selection.
  • Ctrl + Hold left-click + Mouse move: Scroll the selection.
  • In Waveform view: Ctrl + Hold right-click + Mouse move: Stretch the selection from its center.
  • In Phase view or Group Delay view: Hold both Ctrl,Shift and Left-click + Mouse move: Scroll the selection of the waveform.
  • Escape key : Clear any selection.

Playback

  • Space : Play the current selection of the selected file (or the whole file if there is no selection).
  • Shift + Space : Play the selected file while filtering the sound according to the spectral selection.
  • Escape key : Reset any filtering effects.
  • If no selection, put the play cursor at the beginning.

Labels edition

  • Add a label: In Edit mode, simply place the mouse cursor at the time instant where you want to add a label and write (no need to click, select, or anything else).
  • Edit the text of a label: In Edit mode, click in the box of the label you want to edit.
  • Move a label: Drag the label you want to move by holding its vertical line with the left mouse button.
  • Move all labels: Hold Ctrl press left mouse button outside of any specific label and move mouse.
  • Delete a label: Hold the vertical line of the label you want to delete (as if you wanted to move it) and press the delete key.

Waveform edition

  • Grab the waveform wherever and move the mouse up or down to increase or decrease the gain (works also in the amplitude spectrum view).
  • Maintain Ctrl, grab the waveform wherever and move the mouse left or right to delay the waveform.
  • Grab the phase spectrum or group-delay and move it for delaying the waveform.

Estimation

  • Ctrl+F on a sound file for estimating its fundamental frequency (F0). Make a selection in frequency for limiting the range of possible F0 (thus, overriding the default range sets in this setting panel).
  • Ctrl+F on a F0 file for re-estimating it. Make a time selection for limiting the update to the selection.
  • Ctrl+Shift+F force the F0 (re-)estimation disregarding any voice/unvoice decision.
