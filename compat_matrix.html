Compatibility Matrix
Linux
Can run with FFTW3, libsndfile, libsox and SDIF support under 32 and 64-bit machines (tested only on Ubuntu(14.04) and Scientific Linux(6.6)).

on continuous integration: Using FFTW3, libsndfile and 64-bit machine.

Latest updates have to be compiled from GitHub. To facilitate this process, you can run the script distrib/compile_dfasma_from_github.sh, which downloads and compiles the latest updates, or a previous version.

Mac
Can run only with FFTW3 and libsndfile and SDIF support on 64-bit machines (tested only on OSX 10.9).
(since I don't have a Mac, I cannot test it easily)

on continuous integration: Using FFTW3, libsndfile and 64-bit machine (using OSX 10.9).

Latest updates have to be compiled from GitHub (the script mentionned above for Linux might work on OSX).

Windows
Can run only with FFTW3 and libsndfile and SDIF support (tested only on Windows 7(2013)).
MinGW compiles only for 32-bit. MSVC compiles for both 32 and 64-bit.

on continuous integration: Using FFTW3, libsndfile and both 32 and 64-bit machines (using Windows Server 2012 R2).

You can download an installer with the latest updates by following this link; then select your platform; and go under the 'Artifacts' tab on the right.
